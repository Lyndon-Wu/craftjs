## Usage


## Development

To build and view a part:

	bin/craft part [name]

e.g.,

	bin/craft part canvas
	bin/craft part pin


To build and view a design file:

	bin/craft xml [path-to-xml]

e.g.,

	bin/craft xml examples/xml/pins.xml
